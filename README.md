## Feature

- jwt authentication
- RBAC Authority Control (user -> role -> premissions)
- Menu Management (role -> menus)

## Nova front end

[Nova Admin](https://github.com/chansee97/nova-admin)

## Usage

### Installation

```sh
pnpm i
```

### Running

```sh
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
