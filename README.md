## Feature

- jwt authentication
- RBAC Authority Control (user -> role -> premissions)
- Menu Management (role -> menus)

## Usage

### Installation
```
pnpm i
```
### Running
```
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
